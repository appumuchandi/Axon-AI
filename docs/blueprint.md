# **App Name**: AXON-AI

## Core Features:

- Emergency AI Assistant: Interact with an AI-powered tool capable of providing first-aid guidance, disaster survival instructions, emergency recommendations, and safety procedures via text input.
- SOS Emergency System: Trigger emergency alerts, securely share GPS coordinates, and notify predefined emergency contacts with display of emergency status, prioritizing speed and reliability.
- Emergency Profile Management: Create and securely manage a personal emergency profile, storing critical information like full name, blood group, allergies, medical conditions, and emergency contacts, easily accessible during crises.
- Disaster Information Dashboard: A visually clean dashboard displaying critical emergency alerts, connectivity status, battery level, nearby emergency services, and disaster preparedness information.
- AI Insights Engine: Generate AI-powered informational tools for emergency preparedness suggestions, risk awareness tips, disaster response recommendations, and safety improvement guidance.
- Offline Data Persistence: Utilize local data caching for offline access to emergency guidance, user profiles, and critical information to ensure functionality during network failures.
- Secure Data & Access Control: Implement secure user authentication, protect user data, and apply request rate limiting and abuse prevention measures for all stored emergency information.

## Style Guidelines:

- Primary color: Deep Sky Blue (#00BFFF), evoking trust, intelligence, and emergency technology, suitable for key informational elements and branding. (User-provided)
- Background color: Very Light Gray (#F0F0F0), ensuring a modern, clean appearance and high readability, suitable for general UI backgrounds. (User-provided)
- Accent color: Salmon (#FA8072), used strategically for critical elements like SOS buttons, emergency alerts, and important actions to ensure high visibility. (User-provided)
- The font family 'Inter' (sans-serif) will be used throughout the application for a modern, clean, and readable hierarchy with bold headings and consistent spacing. Note: currently only Google Fonts are supported.
- A modern and futuristic interface design that prioritizes simplicity and accessibility during emergencies, fully supporting mobile responsiveness.
- Dashboard layout will organize information clearly using cards, status indicators, clean tabular sections, and prominent emergency action panels.
- Simple, minimal icons will be used, with an emphasis on emergency-themed indicators and intuitive navigation symbols for rapid comprehension.
- Include subtle loading animations, smooth transitions, and lightweight, non-distracting motion effects to enhance user experience without hindering focus in emergencies.
